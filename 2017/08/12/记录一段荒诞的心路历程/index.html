自己的一段荒诞心路历程

发表于 2017-08-12 | 分类于 Invest&Finance | | 阅读次数
投机者要培养一个稳定、良好的心态，在修养和境界上达到高瞻远瞩、超凡脱俗，也许是一辈子的事情。

事件：由于朝鲜和美国之间的紧张局势升级，周四，现货黄金逼近1290美元/盎司关口至两个月高位，涨幅达约1%。美国和朝鲜之间互相的言辞攻击加剧，市场极度恐慌，美三大股指开盘大跌，恐慌指数VIX上涨逾30%，最高突破15，创5月18日以来新高。亚欧股市周四全线下跌，港股一度狂泻450点，创9个月最大跌幅。早盘韩国和沪指股市跌逾1%。欧洲股市全线低开。MSCI全球指数(MSCI All-Country World Index)跌0.2%，至两周多最低水平。投资者涌入黄金，疯狂购买避险资产。日内公布的美国数据偏空，美国7月PPI环比下跌0.1%，为11个月来的首次下跌；表明近一年来的通胀将进一步缓和。美国至8月5日当周初请失业金人数为24.4万人，高于前值和预期。

黄金行情经过了周四一天一夜的暴涨，周五白天三次上涨到1288.85左右受压力位回落，二次下探1283.75左右反弹振荡，多空双方都开始在等候晚8：30美国7月季调后CPI月率公布后明确方向。

分析：

  • [x] 判断一：今天特朗普向朝鲜发动更猛烈的嘴炮，在货币战争威力更强于军事战争的年代，虽发动战争的可能性较小，但恐慌指数蹿升美元美股齐坠，黄金走势将继续看多，今晚数据公布利好可望突破1290美元/盎司，有望上探1300美元/盎司。

  • [x] 决断二：晚8：30美国7月季调后CPI月率公布应该低于预期，将是利多黄金；如果公布出来的数据高于预期，利空黄金昨天的获利盘会引起跳空大跌到一定的点位，不得不防。

    决策：

  • [x] 操作一：晚8：25，数据公布前10分钟，黄金三层仓位提早布局多单，止损二个点。
  • [x] 操作二： 公布数据利多，上破1290，加二层仓继续做多;
  • [x] 操作三： 数据公布利空，反手做空三层仓；再低多。

    荒诞可笑的事发生了！

    能影响行情重要数据晚8：30美国7月季调后CPI月率公布前几分钟，大多数有头寸持仓的交易员内心不够强大，心情是忐忑的，特别是仓位较重的，因为自己的分析判断是跟大众的分析相背而行的。我也一样，在时间慢慢接近公布数据之前 ，除了紧盯数据公布前的一些可捕捉的参考价值信息，开始寻找可以缓减紧张、支撑内心强大的东西。
    笑话就在这一时刻出现了，从微信朋友圈找出了观音菩萨大像的祥和音乐视频，开始放了一遍又一遍听起来。听了越来越纠结，希望世界和平没有战争，现在却内心希望多些利多黄金的数据出来。8：30数据终于公布了，如自己所推测，利多黄金。

    太好了！符合了自己推测，行情来了一波急速大涨，看着行情急速上破了1290位，急忙追多加仓二层指望到1295以上，看着帐户里的美金急剧一直飘多增长一万。。。二万。。。，心中乐开了，当然忘不了一直放着的祥和音乐，连忙跪到三拜磕头感谢观音大士。

    悲催了！ 自顾高兴了，忘了昨天已获利盘高位出跳的可能性风险，在刚上1292.05的位置就来了个急速下跌，还好有设了止损位，却把赢利的损失了大部分，在数据利好的时候才刚刚上破1290，根本没反应过来反手做空。

    被打击了！ 行情开始趋于平衡，坚持自己的低多观点，二次过早低多做单进场到止损点被打掉，把赢利损失还亏损了，第三次低多做单五层仓进场，在一批市场唱空的氛围中坚持看多，却设了较少的止损点位单子被振荡波动中的行情打掉，只把损失赢利了回来。一夜白忙活，几个来回下来，搞得精疲力尽，想周六凌晨就休市（嘿，怎会想不浪费过夜仓费了），早早收场休息，放弃了自己的系统，就是放弃了一次赢利的机会（行情在后来的走势中回调到1290.12美元/盎司）。
    总结：人性本质上难以摆脱的非理性因素，了解自己、战胜自己这个古老的哲学话题，看起来似乎有可能实现，实际上也许永远只是人类追求的理想而已。要培养一个稳定、良好的心态，在修养和境界上达到高瞻远瞩、超凡脱俗，也许是一辈子的事情。
