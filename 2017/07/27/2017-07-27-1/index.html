布林线指标的妙用之处！（图解）

发表于 2017-07-27 | 分类于 Invest&Finance | | 阅读次数
解读布林线

布林线虽然简单，但如何运用却博大精深，简简单单的三条线却能清晰地表现当前市场的状态，到底是上涨还是下跌，是即将进入震荡还是行情即将来临，这让人们充满好奇。笔者将在本文中，为大家详细解读布林线。

布林线原理

我们首先来简单的介绍下布林线的构成。布林线由上、下轨和中轨构成，中轨就是简单移动平均线，一般选用20期，上下轨通过中轨加减2倍标准差计算出来的。标准差是布林线指标核心，其计算公式如下：

布林线的四个状态及其状态转换

布林线中轨有方向，上下轨也有，根据布林线三轨方向地表现，归结出四个不同的状态。

我们总爱用趋势和震荡来描述当前市场所处的状态，如果用布林线的状态来描绘当前市场，收口和走平可以归结震荡市场，开口和三轨同向可以归结为趋势市场。市场总是围绕这个四个状态重复运行，如果我们能洞悉这个四个状态的转换规律，就能预测市场的下一步走势。

##下面就对四大状态间的转换规律进行简单的介绍。

行情总是涨涨停停，当布林线开口之后，市场总要进行修正整理，这就是收口。三轨同向和走平又是如何解读呢，或者说是怎么形成的呢？
秘密在于中轨。当布林线收口后，中轨如能提供支撑或阻力，行情继续运行。如果中轨不能有效的支撑阻力，那么价格会围绕中轨展开震荡整理，继续收口，直至走平。通过下图更能直观的表达上述论点。

搞清楚了布林线四大状态之间的转换关系和衔接关系，对后市预测和具体的交易都大有裨益。无独有偶，酒田战法中提及四大战术指导原则在布林线的四个状态可谓得到完美的体现。
熟悉酒田战法的朋友都知道四句堪称经典的话语。疾如风（及时止损应对假大开），徐如林（在收口阶段高抛低吸），掠如火（持仓拿住，三轨同向），不动如山（走平，尤其是区间过窄，关注有效突破即等待有效大开）。

开口及止损技巧

开口代表着行情快速展开，尽管投资者都厌恶震荡行情，喜好趋势行情，但是行情往往来的很突兀，待发现确认后再进场，由于止损过大不符合盈亏比，难以决定是否追多或追空。
如下图（2012年10月12日15：00黄金走势图）所示：上一小时，黄金快速走低，小时图上布林线开口向下，不过待收盘后再进场做空，在一般情形下，止损需设置在1773的前高上方，需10余美金的止损。
不过在明白了布林线开口原理后，只需设在中轨略上方即可，因为一旦金价回抽时突破中轨中轨，即可确认这波大开是假的，越早出场越好。

其实还有一种更好的判断真假开口的方向，通过更大周期的布林线来判断。还是以上图为例，上图是1小时图20期布林线指标，如果切到5分钟图，同时通过折算使用240期布林线，能更加清晰的观察开口。
同样的开口，在5分钟图上更加情形明了，止损可以设置的更低些，激进者甚至可以设置在240布林线下轨上方一点即可。若价格回到布林线内部则表示大开暂时告一段落，可以考虑获利平仓。

收口及止损技巧

当布林线指标进入收口阶段后，之前的行情走势暂时告一段落，行情将进入反转阶段或暂时回调，如果拥有复合仓位，建议平掉一半仓位。如若后市再度突破中轨，平仓全部仓位为佳。
布林线进入收口阶段后，行情一般会陷入震荡整理，行情波动不大，以短线区间操作为主，止损设置在可见的最高最低点上下方即可。收口阶段是高抛低吸区间操作的良机。

三轨同向及止损技巧

收口后，中轨若能成为有效的支撑阻力，就会出现三轨同向状态。三轨同向是市场行情极为强劲而且稳定的表现，也是最为考验投资者耐心的时候。这种情形也就是酒田战法中的掠如火，牢牢抱住盈利的头寸，中轨成为很好的加仓点，也成为盈利提止损的参考点，直到跌破布林线中轨为止。

走平及止损技巧

当市场陷入震荡整理后，尤其是窄幅震荡整理，布林线就走平。走平是布林线最为好看但是最难把握的行情，因为面临着布林线随时开口的风险。按照酒田战法的战术指导还是以不动如山，观望为好。不过若是此情形发生早盘，可以进行一些刮头皮的操作，止损设置的就比较宽松。正规的操作还是应该耐心的等着布林线的开口方向。
