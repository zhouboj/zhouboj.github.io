2.15 文件和内建函数open()、file()

如何打开文件

handle = open(file_name,access_mode = ‘r’)

file_name变量包含我们希望打开的文件的字符串名字，access_mode中y表示读取，‘W’表示写入，‘a’表示添加。其他可能用到的标识还有“+”表示读写，‘b’表示二进制访问。如果open()成功，一个文件对象返回之后，我们就可以访问它的一些方法，比如readlines()和close()。文件对象的方法属性也必须通过句点属性标识访问 参阅下面的核心笔记^1
